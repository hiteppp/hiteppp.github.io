723_promise问题_能否执行多个回调

发表于 2023-07-23 分类于 ajax-promise-axios 阅读次数： 
<script>
      let p = new Promise((resolve, reject) => {
        resolve("OK");
      });

      //若Promise的状态发生改变，所有的回调函数就会全部执行
      //若不改变，就不会执行

      p.then((value) => {
        console.log("wrong");
      });

      p.then((value) => {
        alert("wrong");
      });
    </script>
